Usando links externos

Para utilziar um link externo, basta utilizar a tag <a href = "Insira aqui o link">.
Ex: Aqui, aprederei a utilizar links externos. Meu exemplo será o GITHUB

Sobreposição de página

Para que sua página não seja sobreposta pelo link externo, é recomendável utilziar os parâmetros target="_blank" e rel="external"
Código: <a href="img/pagina2.html" target="_self" rel="next">PÁGINA IMAGEM DO GATO


Links para uma outra página do mesmo site

PÁGINA IMAGEM DO GATO
Código: <a href="img/pagina2.html" target="_self" rel="next">PÁGINA IMAGEM DO GATO

Como fazer links para download

Para criar um link de download, faz-se assim:
Exemplo: <a href="arquivo/boleto.pdf" download="boleto.pdf" type="application/pdf">Clique para baixar! </a>
Visualmente fica assim: Clique aqui para baixar!

Exemplos com listas!

  1. Boleto.pdf
  2. Boleto.zip

Código:
<ol>
<li> <a href="arquivo/boleto.pdf" download="boleto.pdf" type="application/pdf">Boleto.pdf</a> </li>
<li> <a href="arquivo/boleto.zip" download="boleto.zip" type="application/zip" >Boleto.zip</a> </li>
</ol>
