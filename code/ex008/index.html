Formas de formatação

Negrito / Destaque

Para criar destacar uma parte do texto em negrito, é utilizado a tag <STRONG>.

Itálico / Ênfase

Para deixar uma parte do texto em itálico, é utilizado a tag <EM>.

Texto marcado

Para marcar a parte selecionada do texto, é utilizada a tag <MARK>.
Para mudar a cor do do do marca-texto é utilizado CSS.

Texto pequeno e texto grande

Para deixar a parte selecionada to texto pequena é utilziada a tag <SMALL>.
Agora, para deixar a parte do texto grande, é utilizada a tag <BIG>.
OBS:A tag BIG é uma tag morta e desconsiderata pela W3C, mesmo ainda estando em funcionamento.

Texto deletado

Para riscar uma parte do texto, é utilizado a tag <del>.

Texto sublinhado

Para sublinhar uma parte desejada no texto, é utilziado a tag <INS>.
OBS:Também podemos utilizar a tag U, porém ela é uma tag que não possui semântica, apenas forma.

Texto sobrescrito e subescrito

Para criarmos textos sob e subescritos, utilizaremos as tags <sup> e <sub>.
EX: X2+50 ---------- H2O

Códigos

Para exibir um código em sua página HTML, você utilziará a tag <CODE>.
Exemplo: <code>num = int(input('Digite um número')) if num % 2 == 0: print(f'0 número {num} é ímpar') else: print(f'0 número {num} é ímpar') print('Fim do programa')</code>
Porém, notará que o código ficará completamente bugado. Então, para que seja adicionado a quebra de linha, utilize antes de CODE a tag <pre>.
